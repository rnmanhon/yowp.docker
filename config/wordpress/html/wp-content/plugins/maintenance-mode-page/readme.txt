== Description ==

Maintenance Mode Page Plugins is displays a coming soon page for your wordpress front end for visitors, while displaying the usual website for admin. This plugins is allow to add a stylish coming soon template with background image slideshow, countdown timer, about us page, contact info page and subscribe field. The look stands out and will remain memorable to all of your visitors.

== Installation ==

1. Upload the entire `maintenance-mode-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Maintenance Mode Panels menu to customize this plugin.
4. Finally Enjoy it.
